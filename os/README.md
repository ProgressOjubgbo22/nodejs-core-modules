# Learning Node.js - OS Module

This folder contains my practice examples of Node.js OS module.

## What I am learning
- Getting OS platform using os.platform()
- Getting CPU architecture using os.arch()
- Accessing CPU information using os.cpus()
- Checking total and free memory using os.totalmem() and os.freemem()
- Getting the home directory with os.homedir()
- Checking system uptime with os.uptime()
- Accessing current user info using os.userInfo()
- Getting temporary directory with os.tmpdir()

## Purpose
I am practicing the OS module to understand how to get system information and manage resources in Node.js.
